# Использование переведенных слагов в представлениях

Для обработки переводимых слагов в **DetailView** можно использовать **TranslatableSlugMixin**, чтобы сделать эту работу ровной. Например:

```python
class ArticleDetailView(TranslatableSlugMixin, DetailView):
    model = Article
    template_name = 'article/details.html'
```

**TranslatableSlugMixin** гарантирует, что:

* Объект извлекается в правильном переводе.
* Поле **slug** считывается из модели перевода, а не из общей модели.
* Резервные языки обрабатываются.
* Объекты не отображаются случайно в своих резервных слагах, а перенаправляются на переведенные слаги.
